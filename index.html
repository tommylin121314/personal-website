Projects

Here is what I've been up to.

NBA Hall of Fame Predictor

A binary classifier using a logistic regression model to determine the likelihood of a current NBA player being inducted into the National Basketball Assosciation Hall of Fame. Also studied were correlations and trends between the different features like points per game, accolades, and more.

Learn more about the project on my Github.

Financial Headlines Sentiment Analysis

This was a project done for my natural language processing course. Worked in a small team to train a model that could perform sentiment analysis on market news headlines. FinBERT was used as the pre-trained model and data from kaggle.com was used to fine-tune the model. Our model was able to classify finance headlines to be either positive, neutral, or negative above an accuracy of 90%.

Learn more about the project on my Github.

Cat Eats Meat

Developed a hyper-casual mobile game called "Cat Eats Meat" as a passion project in 2020. Learned about Unity and the C# programming language as I built the game from scratch. Also gained experience bug testing, producing simple music and art, and publishing. The game is available on the Google Play Store.

Check it out on the Google Play Store.
